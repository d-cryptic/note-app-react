# Notes APP

- Built using ReactJS

## Packages, dependencies and installation guide

1. `npx create-react-app ./`
2. Install `react-icons`
   - `npm i react-icons`
3. `npm start`
4. `nanoid` - for generating unique ids
   - `npm i nanoid`
5. The app uses `Local Storage` to store all your notes
